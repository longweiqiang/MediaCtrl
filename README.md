**简要描述：** 

- 网易云PC版控件(2.0版本)

**注意事项：** 
- 该工具依据软件的快捷键(全局快捷键)实现，故网易云软件的快捷键需要保持默认
  
**编译命令：**
- pip install -r requirements.txt

- pyinstaller -Fw MediaCtrl.py

**使用方式：** 
1. 双击dist目录下的install.vbs文件后，桌面会生成四个快捷方式
2. 将快捷方式固定到任务栏中，开始使用吧

**备注：** 
- pre：上一曲
- next：下一曲
- p：播放-暂停
- c：显示/隐藏桌面歌词
